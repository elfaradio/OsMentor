# OsMentor AI

OsMentor AI is an Operating Systems tutoring chatbot built around a production RAG backend.

## Backend Stack

- Python 3.12
- FastAPI
- ChromaDB
- LangChain
- Sentence Transformers
- Gemini API
- PyMuPDF
- Pydantic
- Uvicorn

## Project Layout

- `data/raw/` - place source PDF textbooks here
- `data/extracted/` - page-level JSON generated by ingestion
- `data/chunks/` - chunk JSON generated by the chunker
- `vector_db/chroma/` - persistent ChromaDB store

## Setup

1. Create a virtual environment.
2. Install dependencies from `backend/requirements.txt`.
3. Copy `.env.example` to `.env` and set `GOOGLE_API_KEY`.

## Ingestion Pipeline

Run PDF extraction:

```bash
python -m rag_pipeline.ingest
```

Chunk extracted pages:

```bash
python -m rag_pipeline.chunking
```

Run the full indexing pipeline in one command:

```bash
python -m rag_pipeline.index
```

Embed chunks and load them into Chroma from your application code using `rag_pipeline.embeddings` and `vector_db.chroma_manager`.

## Run the API

```bash
uvicorn backend.app.main:app --reload
```

## Chat Endpoint

`POST /chat`

Request:

```json
{
  "question": "What is paging?"
}
```

Response:

```json
{
  "answer": "...",
  "citations": [
    {
      "source": "silberschatz.pdf",
      "page": 320
    }
  ]
}
```
